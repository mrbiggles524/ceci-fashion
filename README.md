# Ceci's Dream Dress Studio

**Current version:** **v1.0.6**

Live: https://mrbiggles524.github.io/ceci-fashion/

Bump `CECI_APP_VERSION` in `version.js` and every `?v=` in `index.html` on every update.

## Play locally

```powershell
cd side-quests/ceci-fashion
python -m http.server 8900
```

Open http://127.0.0.1:8900/

## How to play

1. Pick a theme: Unicorn, Fairy, or Princess
2. Choose a dress color and pattern
3. Tap a sticker, then tap the dress to place it
4. Drag stickers to move them · Double-click to remove
5. **Surprise Me!** for a random dress · **Save My Dress** to download a picture
